AdCount: A user friendly, automated method to quantify subcellular structures in confocal images

This is the official website where you can download AdCount and read more about how to use it.

About AdCount

AdCount is an image processing suite developed by Aditya Nair, an undergraduate at the National University of Singapore with the supprot and mentorship of Dr Lim Kah Leong and Dr John Thundyil at the National Neuroscience Institute, Singapore

AdCount is unique for its focus on usability and automation for its analysis. The interface used to navigate AdCount is straighforward and easy to use, while the results of its analysis are free from user bias. More information about the algorithms used can be read in the upcoming publication on AdCount

Know more about how to use AdCount using the AdCount User Manual found at this link: AdCount User Manual.
