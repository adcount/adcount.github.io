AdCount: A user friendly, automated method to quantify subcellular structures in confocal images

Aditya Nair, John Thundyil, Lim Kah Leong

This is the official website where you can download AdCount and read more about how to use it.

Download AdCount

AdCount can be downloaded using this link: AdCount 1.1.

TO use AdCount, Wolfram Mathematica must be installed in the PC/Mac. Simply open the AdCount.cdf file and perform your analysis.

DownLoad AdCount-Manual

AdCount-Manual is a special version of AdCount which offers more manual control over image analysis. We reccomend using this version when AdCount fails to capture the required feature faithfully. More information can be seen in the AdCount User Manual available for download below

Know more about how to use AdCount using the AdCount User Manual found at this link: AdCount User Manual.
