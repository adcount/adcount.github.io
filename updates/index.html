AdCount: A user friendly, automated method to quantify subcellular structures in confocal images

This is the official website where you can download AdCount and read more about how to use it.


Current Version: AdCount 1.0

Upcoming Versions AdCount is an image processing suite developed in the language Wolfram Mathematica.
Its current applications include the automated analysis of lipid droplets and mitochondria in two dimensional fluorescence confocal microscope images.

As AdCount uses Wolfram Mathematica to perform its computations, this software must first be installed for AdCount to be functional. Mathematica Student is free to download for NUS Staff and Students.

AdCount's uniqueness lies in its simple and easy to use interface.


Know more about how to use AdCount using the AdCount User Manual found at this link: AdCount User Manual.
