AdCount: A user friendly, automated method to quantify subcellular structures in confocal images

Aditya Nair, John Thundyil, Lim Kah Leong

This is the official website where you can download AdCount and read more about how to use it.

Current Version

AdCount 1.0

This version can perform automated quantification of lipid droplets, mitochondria and neurites in two dimensional confocal microscopy images. Features such as area, length and perimeter can be obtained using this version

AdCount 1.1

This version provides better stability and bug fixes

Upcoming Versions

The next version will feature the ability to analyze three dimensional confocal microscopy images

Know more about how to use AdCount using the AdCount User Manual found at this link: AdCount User Manual.
